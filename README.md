# PayWay Checkout - Node.js Integration

A complete Node.js implementation of ABA PayWay payment gateway integration.

## Features

- ✅ Express.js server with EJS templating
- ✅ Secure HMAC-SHA512 hash generation
- ✅ PayWay checkout form integration
- ✅ Customizable customer data
- ✅ Success/Failure payment pages
- ✅ Responsive design
- ✅ Environment configuration
- ✅ Error handling
- ✅ Development debug mode
